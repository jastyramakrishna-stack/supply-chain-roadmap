📘 Supply Chain Roadmap
A visual, interactive roadmap to learn Supply Chain Management from fundamentals to advanced planning, forecasting, replenishment, analytics, optimization, and modern digital supply chain technologies.
Built as a lightweight website using React (via CDN) and deployed on GitHub Pages.

🚀 Overview
This roadmap provides a structured learning path for students, professionals, and teams looking to build strong supply chain capabilities.
It covers end‑to‑end supply chain concepts, tools, models, and modern digital practices in a clean, searchable UI.

📦 Features
Interactive roadmap UI

Covers basics → advanced supply chain topics

Includes forecasting models & replenishment logic

Search bar for quick topic lookup

Lightweight (no build tools or dependencies)

Mobile‑friendly responsive layout

Easy to extend via data.js

🌐 Deployment
This project is designed for GitHub Pages.

Push the repo to GitHub

Go to Settings → Pages

Select:

Branch: main

Folder: /root

Save

Your site will be live at:

Code
https://your-username.github.io/supply-chain-roadmap/
